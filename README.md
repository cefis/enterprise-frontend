Projeto front-end criado em Nextjs 14 e TailwindCss para o processo seletivo da empresa Cefis

Para iniciar o projeto


```bash
npm install
# or
yarn install
# or
pnpm install
```

```bash
npm run dev
# or
yarn run dev
# or
pnpm run dev
```

Abra http://localhost:3000 no seu navegador para ver o resultado.
